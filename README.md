# -Python minesweeper.py-
マインスイーパー　ゲーム作成（Python）

# マインスイーパー 開発構成図

## 1. 盤面生成

* 盤面サイズ設定

  * 8 × 8
* 空盤面作成
* 地雷配置

  * ランダム座標生成
  * 地雷数設定
* 内部盤面作成

```
board
┌─────────────┐
│ 0 0 0 0 0 │
│ 0 9 0 0 0 │
│ 0 0 0 0 0 │
└─────────────┘

9 = 地雷
```

---

## 2. 数字計算

* 各マスを走査
* 周囲8マス確認
* 地雷数をカウント
* 数字を格納

```
1 1 1
1 9 1
1 1 1
```

---

## 3. プレイヤー表示盤面

* プレイヤー用盤面作成
* 初期状態は全マス非表示

```
■ ■ ■ ■
■ ■ ■ ■
■ ■ ■ ■
```

---

## 4. 入力処理

* x座標入力
* y座標入力
* 入力値チェック

  * 範囲内か
  * 既に開いていないか

---

## 5. マスを開く

### 地雷の場合

```
Game Over
```

### 数字の場合

* 数字を表示

### 0の場合

* 周囲のマスを自動展開
* 再帰処理またはキュー探索

---

## 6. 盤面表示

* 開いたマスを表示
* 閉じたマスは ■ のまま

例

```
■ ■ ■ ■
■ 1 1 ■
■ 1 9 ■
```

---

## 7. 勝利判定

* 地雷以外を全て開いたか確認

```
開いたマス数
=
全マス数 - 地雷数
```

達成時

```
You Win!
```

---

## 8. ゲームループ

```
開始
 ↓
盤面生成
 ↓
数字計算
 ↓
表示
 ↓
入力
 ↓
マスを開く
 ↓
勝利判定
 ↓
終了？
 ├─ Yes → 終了
 └─ No
      ↓
    表示へ戻る
```
