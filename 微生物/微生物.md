## 問題1　日和見感染の増加に関連する要因でないのはどれか。

1. 長期臥床の高齢者の増加
2. 免疫抑制療法の増加
3. 抗生薬の使用量の不足
4. 医療従事者を介した感染

**回答：3**

**解説**

1. 長期臥床の高齢者の増加  
   → 免疫力低下や基礎疾患により、日和見感染が起こりやすい。

2. 免疫抑制療法の増加  
   → 免疫機能が抑えられ、日和見感染のリスクが高まる。

3. 抗生薬の使用量の不足  
   → 正しい。日和見感染の増加には、抗生薬の「過剰使用」や菌交代現象などが関係しやすく、使用量の不足は関連しない。

4. 医療従事者を介した感染  
   → 院内感染の原因となり、日和見感染の増加に関与する。

## 問題2　日和見感染症の起炎菌はどれか。2つ選べ。【第103回】

1. メチシリン耐性黄色ブドウ球菌〈MRSA〉
2. インフルエンザ菌
3. A群溶連菌
4. 髄膜炎菌
5. 緑膿菌

**回答：1、5**

**解説**

1. メチシリン耐性黄色ブドウ球菌〈MRSA〉  
   → 免疫低下時に感染を起こしやすい代表的な日和見感染菌である。

2. インフルエンザ菌  
   → 健常者にも感染する一般的な病原菌である。

3. A群溶連菌  
   → 日和見感染ではなく、一般感染症の起炎菌である。

4. 髄膜炎菌  
   → 主に健常者にも急性発症する感染症の原因菌である。

5. 緑膿菌  
   → 免疫低下患者で問題となる代表的な日和見感染菌である。

## 問題3　ウイルスの一般的特徴について正しいのはどれか。【第88回】

1. 大きさは1〜10μmである。
2. 増殖に生きた細胞が必要である。
3. アルコール消毒は無効である。
4. 抗生薬によって不活化される。

**回答：2**

**解説**

1. 大きさは1〜10μmである。  
   → 誤り。ウイルスは細菌より小さく、nm単位である。

2. 増殖に生きた細胞が必要である。  
   → 正しい。ウイルスは自己増殖できず、宿主細胞が必要である。

3. アルコール消毒は無効である。  
   → 誤り。多くのウイルスにアルコール消毒は有効である。

4. 抗生薬によって不活化される。  
   → 誤り。抗生薬は細菌に対する薬であり、ウイルスには無効である。

## 問題4　感染症の予防及び感染症の患者に対する医療に関する法律〈感染症法〉において、重症急性呼吸器症候群〈SARS：Severe Acute Respiratory Syndrome〉の分類はどれか。

1. 一類感染症
2. 二類感染症
3. 三類感染症
4. 四類感染症

**回答：2**

**解説**

1. 一類感染症  
   → エボラ出血熱などが分類される。

2. 二類感染症  
   → 正しい。SARSは二類感染症に分類される。

3. 三類感染症  
   → 腸管出血性大腸菌感染症などが分類される。

4. 四類感染症  
   → 動物・昆虫などを介して感染する感染症が中心である。

## 問題5　感染症と保健所への届出期間の組み合わせで正しいのはどれか。

1. 結核：診断後7日以内
2. 梅毒：診断後直ちに
3. E型肝炎：診断後直ちに
4. 腸管出血性大腸菌感染症：診断後7日以内
5. 後天性免疫不全症候群〈AIDS〉：診断後直ちに

**回答：2、3、5**

**解説**

1. 結核：診断後7日以内  
   → 誤り。結核は診断後、直ちに届出が必要である。

2. 梅毒：診断後直ちに  
   → 正しい。

3. E型肝炎：診断後直ちに  
   → 正しい。

4. 腸管出血性大腸菌感染症：診断後7日以内  
   → 誤り。腸管出血性大腸菌感染症は診断後、直ちに届出が必要である。

5. 後天性免疫不全症候群〈AIDS〉：診断後直ちに  
   → 正しい。
