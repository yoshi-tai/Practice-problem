## 問題1　インフルエンザが流行しているが、小規模多機能型居宅介護を行う事業所では罹患者はいない。この事業所で看護師が行う罹患予防の対策で最も適切なのはどれか。

1. 宿泊の利用を断る。
2. 湿度を10%以下に保つ。
3. 利用者に手洗いを勧める。
4. 利用者に予防的に抗インフルエンザ薬を与薬する。

**回答：3**

**解説**

1. 宿泊の利用を断る。  
   → 罹患者がいない状況での感染予防策としては不適切。

2. 湿度を10%以下に保つ。  
   → 乾燥によりインフルエンザがむしろ拡大しやすくなる。

3. 利用者に手洗いを勧める。  
   → 最も基本的で有効な感染予防策である。

4. 利用者に予防的に抗インフルエンザ薬を与薬する。  
   → 通常は医師の判断が必要であり、看護師の判断で予防的に与薬するものではない。

## 問題2　循環式浴槽の水質汚染によって発生するのはどれか。

1. B型肝炎
2. マラリア
3. レジオネラ肺炎
4. 後天性免疫不全症候群〈AIDS〉

**回答：3**

**解説**

1. B型肝炎  
   → 血液・体液感染であり、水では感染しない。

2. マラリア  
   → 蚊が媒介する感染症であり、浴槽では感染しない。

3. レジオネラ肺炎  
   → 循環式浴槽、温泉、加湿器などの汚染水が主な原因となる。

4. 後天性免疫不全症候群〈AIDS〉  
   → HIVによる血液・体液感染であり、水では感染しない。

## 問題3　空気感染するのはどれか。

1. 結核菌
2. 腸管出血性大腸菌
3. ヒト免疫不全ウイルス〈HIV〉
4. メチシリン耐性黄色ブドウ球菌〈MRSA〉

**回答：1**

**解説**

1. 結核菌  
   → 空気感染する。

2. 腸管出血性大腸菌  
   → 経口感染する。

3. ヒト免疫不全ウイルス〈HIV〉  
   → 血液・体液感染する。

4. メチシリン耐性黄色ブドウ球菌〈MRSA〉  
   → 主に接触感染する。

## 問題4　誤嚥で発症するのはどれか。

1. 肺炎
2. 胃炎
3. 肝炎
4. 膵炎

**回答：1**

**解説**

1. 肺炎  
   → 誤嚥により発症する。代表例は誤嚥性肺炎である。

2. 胃炎  
   → 誤嚥とは直接関係しない。

3. 肝炎  
   → 誤嚥とは直接関係しない。

4. 膵炎  
   → 誤嚥とは直接関係しない。

## 問題5　肺結核について正しいのはどれか。2つ選べ。

1. 結核の10%程度である。
2. 感染経路は接触感染である。
3. 肺アスペルギルス症と同じ原因菌である。
4. 二次性の発症は過去の感染の再活性化による。
5. DOTS〈Directly Observed Treatment, Short-course〉が推奨される。

**回答：4、5**

**解説**

1. 結核の10%程度である。  
   → 誤り。結核の多くは肺結核であり、最も一般的である。

2. 感染経路は接触感染である。  
   → 誤り。肺結核は空気感染する。

3. 肺アスペルギルス症と同じ原因菌である。  
   → 誤り。結核は結核菌、アスペルギルス症は真菌であるアスペルギルスが原因である。

4. 二次性の発症は過去の感染の再活性化による。  
   → 正しい。

5. DOTS〈Directly Observed Treatment, Short-course〉が推奨される。  
   → 正しい。服薬支援を伴う結核治療の基本である。

## 問題6　日本の結核について正しいのはどれか。2つ選べ。

1. 一類感染症に分類されている。
2. 公費で医療費を負担する制度がある。
3. 令和5年〈2023年〉の死因別死亡数上位5位以内である。
4. DOTS〈Directly Observed Treatment, Short-course〉が推奨される。
5. ツベルクリン反応検査を実施してからBCG接種を行うことが定められている。

**回答：2、4**

**解説**

1. 一類感染症に分類されている。  
   → 誤り。結核は二類感染症に分類されている。

2. 公費で医療費を負担する制度がある。  
   → 正しい。結核医療には公費負担制度がある。

3. 令和5年〈2023年〉の死因別死亡数上位5位以内である。  
   → 誤り。結核は死因上位5位以内には入らない。

4. DOTS〈Directly Observed Treatment, Short-course〉が推奨される。  
   → 正しい。服薬支援による治療が基本である。

5. ツベルクリン反応検査を実施してからBCG接種を行うことが定められている。  
   → 誤り。現在は生後1歳未満への一律接種であり、ツベルクリン反応検査は前提ではない。

## 問題7　令和4年〈2022年〉の日本の結核対策で増加が問題とされているのはどれか。  

【第108回改変 午前33問】

1. 新登録結核患者数
2. 喀痰塗抹陽性の肺結核患者数
3. 活動性全結核患者数
4. 新登録結核患者における80歳以上の割合

**回答：4**

**解説**

1. 新登録結核患者数  
   → 減少傾向である。

2. 喀痰塗抹陽性の肺結核患者数  
   → 減少傾向である。

3. 活動性全結核患者数  
   → 全体としては減少傾向である。

4. 新登録結核患者における80歳以上の割合  
   → 正しい。増加が問題とされている。

高齢者の割合が年々増加しており、高齢化結核が課題となっている。

## 問題8　飛沫感染するのはどれか。

1. 疥癬〈かいせん〉
2. コレラ
3. A型肝炎
4. インフルエンザ

**回答：4**

**解説**

1. 疥癬〈かいせん〉  
   → 接触感染する。

2. コレラ  
   → 汚染水や食物による経口感染である。

3. A型肝炎  
   → 経口感染、糞口感染である。

4. インフルエンザ  
   → 咳やくしゃみによって拡散する飛沫感染である。
