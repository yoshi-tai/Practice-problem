## 問題1　女性の骨盤腔内器官について、腹側から背側への配列で正しいのはどれか。

1. 尿道 - 膣 - 肛門管
2. 膣 - 尿道 - 肛門管
3. 肛門管 - 膣 - 尿道
4. 尿道 - 肛門管 - 膣
5. 膣 - 肛門管 - 尿道

**回答：1**

**解説**

女性の骨盤内で、前方から後方へ並ぶ順番は **尿道 → 膣 → 肛門管** である。

## 問題2　排卵のある正常な月経周期で正しいのはどれか。

1. 黄体は形成後1週間で萎縮する。
2. エストロゲンの作用で子宮内膜が分泌期になる。
3. 発育した卵胞の顆粒膜細胞からプロゲステロンが分泌される。
4. エストロゲンのポジティブフィードバックによって黄体形成ホルモンの分泌が増加する。

**回答：4**

**解説**

1. 黄体は形成後1週間で萎縮する。  
   → 誤り。黄体は形成後、約2週間で退縮する。

2. エストロゲンの作用で子宮内膜が分泌期になる。  
   → 誤り。子宮内膜を分泌期にするのはプロゲステロンである。

3. 発育した卵胞の顆粒膜細胞からプロゲステロンが分泌される。  
   → 誤り。プロゲステロンを主に分泌するのは黄体である。

4. エストロゲンのポジティブフィードバックによって黄体形成ホルモンの分泌が増加する。  
   → 正しい。エストロゲンが高値になると、ポジティブフィードバックによりLHサージが起こる。

## 補足問題2　性周期とホルモンについて正しいのはどれか。

1. 増殖期は基礎体温が上昇する。
2. プロラクチンによって排卵が起こる。
3. プロゲステロンは子宮内膜の増殖を促進する。
4. 排卵直前に黄体形成ホルモン〈LH〉が高くなる。

**回答：4**

**解説**

1. 増殖期は基礎体温が上昇する。  
   → 誤り。増殖期はエストロゲン作用下で、体温は低温期のままである。上昇するのは排卵後の黄体期である。

2. プロラクチンによって排卵が起こる。  
   → 誤り。排卵を起こすのはLHサージである。プロラクチンは乳汁分泌に関与する。

3. プロゲステロンは子宮内膜の増殖を促進する。  
   → 誤り。子宮内膜の増殖を促すのはエストロゲンである。プロゲステロンは分泌期をつくる。

4. 排卵直前に黄体形成ホルモン〈LH〉が高くなる。  
   → 正しい。排卵直前にエストロゲンのポジティブフィードバックでLHが急上昇する。

## 問題3　性周期が規則的で健常な成人女性において、着床が起こる時期に血中濃度が最も高くなるホルモンはどれか。

1. アルドステロン

4. 黄体形成ホルモン〈LH〉

**回答：2**

**解説**

   → 性周期とは直接関連しない。

2. プロゲステロン

3. エストラジオール  
   → 排卵直前にピークとなり、着床期が最高ではない。

4. 黄体形成ホルモン〈LH〉  
   → 排卵前に上昇し、着床期には高くない。

5. 卵胞刺激ホルモン〈FSH〉  
   → 排卵前に上昇し、着床期には高くない。

## 問題4　閉経について正しいのはどれか。

1. 閉経すると膣の自浄作用が低下する。
2. 閉経後はエストロゲン分泌が増加する。
3. 日本人の閉経の平均年齢は55歳である。
4. 10か月の連続した無月経の確認で診断される。

**回答：1**

**解説**

1. 閉経すると膣の自浄作用が低下する。  
   → 正しい。エストロゲン低下により膣の自浄作用は弱くなる。

2. 閉経後はエストロゲン分泌が増加する。  
   → 誤り。閉経後はエストロゲンが大きく減少する。

3. 日本人の閉経の平均年齢は55歳である。  
   → 誤り。日本人女性の平均閉経年齢は約50〜51歳である。

4. 10か月の連続した無月経の確認で診断される。  
   → 誤り。閉経の診断は12か月、つまり1年間の無月経が基準である。

## 問題5　広汎子宮全摘出術を受けた患者への退院後の生活に関する説明で正しいものはどれか。

1. 術後2週から性交は可能です。
2. 定期的に排尿を試みてください。
3. 調理の時は手袋をしてください。
4. 退院当日から浴槽の湯につかることができます。

**回答：2**

**解説**

1. 術後2週から性交は可能です。  
   → 誤り。性交は術後約6〜8週は避ける必要があり、2週では早すぎる。

2. 定期的に排尿を試みてください。  
   → 正しい。広汎子宮全摘出術後は排尿障害が起こりやすいため、定期的に排尿を試みる指導が必要である。

3. 調理の時は手袋をしてください。  
   → 誤り。調理で特別に手袋をする必要はない。

4. 退院当日から浴槽の湯につかることができます。  
   → 誤り。創部感染予防のため、退院当日の入浴、特に湯船につかることは避ける。

## 問題6　機能性月経困難症と診断されたAさんの下腹部痛について、説明で適切なのはどれか。

Aさん〈17歳、高校生〉は身長158cm、体重48kg。最近、月経時に下腹部痛が繰り返し出現し、寝込むことが多くなった。心配した母親と一緒に産婦人科クリニックを受診し、医師から機能性月経困難症と診断された。既往歴に特記すべきことはない。

1. プロスタグランジンの過剰産生によって起こる。
2. 無排卵性の月経によって起こる。
3. 卵巣内のうっ血によって起こる。
4. 経血の流出によって起こる。

**回答：1**

**解説**

1. プロスタグランジンの過剰産生によって起こる。  
   → 正しい。プロスタグランジンが過剰に産生されると子宮収縮が強くなり、下腹部痛を引き起こす。

2. 無排卵性の月経によって起こる。  
   → 誤り。無排卵月経では痛みが起こりにくく、機能性月経困難症の主因ではない。

3. 卵巣内のうっ血によって起こる。  
   → 誤り。卵巣のうっ血は主因ではない。

4. 経血の流出によって起こる。  
   → 誤り。経血の流出そのものが痛みの原因ではない。

## 問題7　機能性月経困難症のAさんへの日常生活指導で適切なのはどれか。

Aさんは「次の月経からは、痛みがあれば処方された鎮痛薬を飲むようにします。部活動で毎日ジョギングをしていますが、その他に日常生活で気をつけることを教えてください」と看護師に聞いてきた。

   → 正しい。血流がよくなり、痛みの緩和に役立つ。

3. 葉酸を含む食品は控える。  
   → 誤り。月経困難症とは無関係で、控える必要はない。

4. 腰部のマッサージは控える。  
   → 誤り。血流改善に役立つこともあり、控える必要はない。

## 問題10　萎縮性膣炎に伴う状態について正しいのはどれか。

1. 性交痛
2. 白色の帯下
3. 膣壁の肥厚化

2. 白色の帯下  
   → 誤り。萎縮性膣炎では帯下は減少し、乾燥しやすい。

3. 膣壁の肥厚化  
   → 誤り。エストロゲン低下により膣壁は萎縮し、薄くなる。

   → 誤り。エストロゲン減少により自浄作用は低下する。

5. エストロゲン分泌の増加  
   → 誤り。萎縮性膣炎はエストロゲン低下で起こる。

## 問題11　ヒトパピローマウイルス〈HPV〉検査の説明で正しいのはどれか。

1. 子宮頸部の細胞をこすり取って検査します。
2. HPVワクチンを接種した人が対象です。
3. 陽性であれば子宮頸がんと診断されます。
4. HPV抗原検査も同時に行います。

**回答：1**

**解説**

1. 子宮頸部の細胞をこすり取って検査します。  
   → 正しい。細胞を採取してHPV感染の有無を調べる。

2. HPVワクチンを接種した人が対象です。  
   → 誤り。接種の有無に関係なく検査は行われる。

3. 陽性であれば子宮頸がんと診断されます。  
   → 誤り。陽性はウイルス感染を示すものであり、がんとは限らない。

4. HPV抗原検査も同時に行います。  
   → 誤り。通常は細胞診またはHPV DNA検査が一般的である。

## 問題14　経腟超音波検査の説明で正しいのはどれか。

Aさん〈47歳、女性、会社員〉は夫〈54歳〉と二人暮らし。6か月前から月経不順になり、閉経前の症状と思い様子を見ていた。しかし、徐々に普段の月経時の出血量よりも多くなり、下腹部痛が出現してきたため、病院の婦人科外来を受診した。診察後、経腟超音波検査の指示が出され、看護師はAさんに検査について説明することになった。

## 問題18　乳癌術後の左上腕のケア方法で正しいのはどれか。2つ選べ。

Aさん〈65歳、女性〉は5年前に乳癌の左胸筋温存乳房切除術と左腋窩リンパ節郭清術を受けた。1年前に大腿骨転移のため日常生活動作〈ADL〉に一部介助が必要となり、訪問看護を利用し在宅で療養している。Aさんの左上腕内側の皮膚をつまむと、健側より厚みがある。

1. 指圧する。
2. 皮膚の露出は少なくする。
3. 保湿クリームを塗布する。
4. ナイロン製タオルで洗う。
5. アルカリ性石鹸で洗浄する。

**回答：2、3**

**解説**

1. 指圧する。  
   → 誤り。リンパ浮腫を悪化させる可能性があるため行わない。

2. 皮膚の露出は少なくする。  
   → 正しい。寒冷や外傷を避け、感染予防として適切である。

3. 保湿クリームを塗布する。  
   → 正しい。乾燥予防になり、リンパ浮腫ケアとして適切である。

4. ナイロン製タオルで洗う。  
   → 誤り。刺激が強く、皮膚トラブルの原因になる。

5. アルカリ性石鹸で洗浄する。  
   → 誤り。皮膚バリアを壊す可能性がある。

## 国家試験過去問　第101回　血栓が存在することによって肺塞栓症を引き起こす可能性があるのはどれか。

1. 右心室
2. 左心房
3. 腎動脈
4. 上大静脈
5. 大腿静脈

**回答：5**

**解説**

1. 右心室  
   → 血栓があれば肺動脈へ流れ肺塞栓になる可能性はあるが、血栓の主な発生部位としては下肢深部静脈が重要である。

2. 左心房  
   → 心原性塞栓の原因になることがあるが、肺塞栓症の主な原因ではない。

3. 腎動脈  
   → ここに血栓ができても肺塞栓症の原因にはなりにくい。

4. 上大静脈  
   → 血栓は比較的生じにくく、肺塞栓症の典型的な塞栓源ではない。

5. 大腿静脈  
   → 正しい。下肢深部静脈血栓症〈DVT〉は肺塞栓症の主原因である。

## 国家試験過去問　第99回　激しい頭痛と嘔吐の後に意識を消失した患者で考えられるのはどれか。

54歳の女性。激しい頭痛と嘔吐の後、意識を消失したため搬入された。呼吸数12/分、不規則。脈拍50/分。血圧210/120mmHg。瞳孔不同がみられる。

1. 一過性脳虚血
2. 脳ヘルニア
3. てんかん
4. 片頭痛

**回答：2**

**解説**

1. 一過性脳虚血  
   → 一過性であれば症状は数分〜数時間で改善することが多く、瞳孔不同や高度な高血圧は説明しにくい。

2. 脳ヘルニア  
   → 正しい。激しい頭痛・嘔吐の後に意識消失、不整な呼吸、徐脈、高血圧、瞳孔不同・固定がみられ、脳ヘルニアを示唆する。

3. てんかん  
   → 一過性の痙攣や意識障害はあり得るが、持続的な高血圧や瞳孔固定とは一致しにくい。

4. 片頭痛  
   → 激しい頭痛はあり得るが、意識消失、瞳孔不同、高度血圧などは典型的ではない。

## 第105回　頭部CTの出血部位について正しいのはどれか。

※原文に「写真を貼り付けられなかったから答えが不明」とあります。解説内容に基づき整理しています。

1. 皮下組織
2. 硬膜外腔
