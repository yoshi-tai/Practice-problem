## 小児の在宅看護・災害看護

14. 障害のレベルを運動機能と知能指数で区分するのはどれか。
1. 大島分類
2. NYHA分類
3. 国際生活機能分類（ICF）
4. Hugh-Jones分類

回答番号 1
1. 大島分類は重症心身障害の判定に、運動機能と知能指数を用いる。
2. NYHA分類は心機能分類である。
3. ICFは生活機能・障害・健康を包括的に分類する枠組みである。
4. Hugh-Jones分類は呼吸困難の程度を示す分類である。

15. Aちゃん（1歳2か月、脳性麻痺、経鼻経管栄養、日常的な口腔内吸引が必要）への経管栄養法で適切なのはどれか。
1. 胃管挿入の長さは、鼻尖から胸骨剣状突起までの直線距離とする。
2. 胃管挿入後は、注射器で空気を1mL注入して気泡音を確認する。
3. 栄養剤を注入する前には毎回、胃内容物が吸引できるか確認する。
4. 栄養剤を注入する際の姿勢は、仰臥位とする。

回答番号 3
1. 胃管長は通常、鼻尖から耳朶を経て剣状突起までを目安にする。
2. 気泡音だけでは胃内留置の確認として不十分である。
3. 注入前に胃内容物の吸引などでチューブ位置や残留を確認する。
4. 誤嚥予防のため仰臥位ではなく上体を挙上する。

55. 小児のアレルギーについて正しいものはどれか。
1. 乳幼児の食物アレルギーで最も多い原因食物は小麦である。
2. 小児気管支喘息で最も多いアレルゲンはスギ花粉である。
3. 小児喘息で急性発作が最も好発する季節は春期である。
4. 川崎病の治療目標は、冠動脈瘤を発生させないようにすることである。

回答番号 4
1. 乳幼児の食物アレルギーでは鶏卵が多い。
2. 小児喘息ではダニやハウスダストが主要アレルゲンである。
3. 発作は秋など季節変動があり、春期が最も典型とはいえない。
4. 川崎病では冠動脈瘤など冠動脈病変の予防が重要な治療目標である。

56. I型からIV型アレルギーの代表的な疾患をそれぞれ2つ述べてください。
1. I型：気管支喘息、食物アレルギー。
2. II型：特発性血小板減少性紫斑病（ITP）、血液型不適合輸血。
3. III型：全身性エリテマトーデス（SLE）、急性糸球体腎炎。
4. IV型：接触性皮膚炎、ツベルクリン反応。

回答番号 記述
1. I型はIgEが関与する即時型反応である。
2. II型は抗体が細胞表面抗原に反応する細胞傷害型である。
3. III型は免疫複合体が関与する。
4. IV型はT細胞が関与する遅延型反応である。

## 感染症

57. Koplik（コプリック）斑がみられる疾患はどれか。
1. 麻疹
2. 手足口病
3. 帯状疱疹
4. ヘルパンギーナ

回答番号 1
1. コプリック斑は麻疹の特徴的所見である。
2. 手足口病では手掌・足底・口腔内の発疹がみられる。
3. 帯状疱疹では神経支配領域に水疱が出る。
4. ヘルパンギーナでは咽頭・口腔内の水疱や潰瘍がみられる。

58. 妊娠初期の感染で児に難聴が生じる可能性が高いのはどれか。
1. 水痘
2. 風疹
3. 麻疹
4. 流行性耳下腺炎

回答番号 2
1. 水痘も先天感染の問題はあるが、先天性難聴として代表的なのは風疹である。
2. 妊娠初期の風疹感染は先天性風疹症候群を起こし、難聴の原因となる。
3. 麻疹は母体合併症などが問題となるが、難聴の代表ではない。
4. 流行性耳下腺炎は後天性難聴の原因となりうるが、妊娠初期感染による児の難聴としては風疹が代表である。

59. 麻疹に関して正しいのはどれか。2つ選べ。
1. 合併症として脳炎がある。
2. 感染力は発疹期が最も強い。
3. 効果的な抗ウイルス薬がある。
4. 2回のワクチン定期接種が行われている。
5. エンテロウイルスの感染によって発症する。

回答番号 1、4
1. 麻疹では肺炎や脳炎などの合併症がある。
2. 感染力は発疹前後に強いが、選択肢中では誤りとして扱われる。
3. 麻疹に特異的で効果的な抗ウイルス薬は一般的ではない。
4. 麻疹風疹混合ワクチンなどで2回定期接種が行われる。
5. 麻疹は麻疹ウイルスによる。

60. 麻疹と診断された1歳男児が細気管支炎を併発し付き添い入院した。入院時、母親への説明で適切なのはどれか。
1. 面会は解熱するまで制限する。
2. 発疹が消失するまで清拭は控える。
3. 含嗽して口腔粘膜を保清する。
4. 個室に入室する。

回答番号 4
1. 感染対策は解熱だけで判断しない。
2. 清潔保持は状態に合わせて行う。
3. 1歳児では含嗽は困難である。
4. 麻疹は空気感染対策が必要で、個室管理が適切である。

61. 風疹について正しいものはどれか。2つ選べ。
1. ウイルスによる感染である。
2. ほとんどの場合、乳児期に発症する。
3. 空気感染によって感染する。
4. 潜伏期間は2～3週間である。

回答番号 1、4
1. 風疹は風疹ウイルスによる感染症である。
2. 乳児期に限って多いとはいえない。
3. 主な感染経路は飛沫感染である。
4. 潜伏期間はおおむね2～3週間である。

62. 風疹の合併症をあげてください。
1. 特発性血小板減少性紫斑病。
2. 脳炎。
3. 関節炎。
4. 妊娠初期感染では先天性風疹症候群。

回答番号 記述
1. 血小板減少性紫斑病は風疹の合併症として知られる。
2. 脳炎はまれだが重要な合併症である。
3. 年長児や成人では関節症状がみられることがある。
4. 胎児への影響として先天性風疹症候群が重要である。

63. 風疹について正しいものはどれか。
1. 発疹は四肢から体幹、顔の順に出現する。
2. 発疹は赤くてかゆみを伴う。
3. 発症すると白血球が急激に増加する。
4. 急性期は水分の摂取を制限する。

回答番号 2
1. 風疹の発疹は顔から体幹・四肢へ広がることが多い。
2. 淡紅色の発疹で、かゆみを伴うことがある。
3. 白血球は減少傾向を示すことがある。
4. 水分制限は通常必要ない。

64. 水痘の発疹の変化として、2番目に起こるのはどれか。
1. 痂皮
2. 丘疹
3. 紅斑
4. 水疱

回答番号 2
1. 痂皮は後期にみられる。
2. 水痘の発疹は紅斑、丘疹、水疱、痂皮へ変化する。
3. 紅斑は最初にみられる。
4. 水疱は丘疹の次にみられる。

65. 水痘について誤っているものはどれか。
1. 潜伏期間は4～7日である。
2. ワクチンは生ワクチンである。
3. 2～10歳頃に好発する。
4. 空気感染が起こる。

回答番号 1
1. 水痘の潜伏期間は一般に約2週間であり、4～7日は短すぎる。
2. 水痘ワクチンは生ワクチンである。
3. 小児に多い感染症である。
4. 空気感染・飛沫感染・接触感染に注意する。

66. 水痘の合併症を2つあげてください。
1. 細菌性皮膚感染症。
2. 膿痂疹。
3. 肺炎。
4. 脳炎。

回答番号 記述
1. 掻破により皮膚の二次感染を起こすことがある。
2. 膿痂疹は皮膚二次感染としてみられる。
3. 肺炎は重症合併症として注意する。
4. 脳炎など中枢神経合併症もまれに起こる。

67. 溶連菌感染後急性糸球体腎炎で入院した7歳女児。入院時の看護計画で適切なのはどれか。
1. 水分摂取を促す。
2. 1日3回の血圧測定を行う。
3. 食事の持ち込みを許可する。
4. 腰部に消炎鎮痛薬を貼用する。

回答番号 2
1. 浮腫や高血圧があるため水分は指示に従って管理する。
2. 急性糸球体腎炎では高血圧や脳症に注意し、血圧測定が重要である。
3. 塩分・水分管理が必要なため自由な持ち込みは不適切である。
4. 腎炎の看護計画として優先されない。

68. 入院3日、頭痛・悪心、血圧148/88mmHg。起こりうる症状はどれか。
1. 眼のかゆみ
2. 意識障害
3. 耳漏
4. 鼻閉

回答番号 2
1. 眼のかゆみは高血圧性の合併症として典型ではない。
2. 急性糸球体腎炎で高血圧が進むと高血圧性脳症による意識障害に注意する。
3. 耳漏は関連が低い。
4. 鼻閉は関連が低い。

69. 入院1週、床上安静が続きイライラしている7歳女児への発言で適切なのはどれか。
1. 「すぐに退院できるから頑張ろう」
2. 「好きなだけテレビや動画を観ていいよ」
3. 「ベッドに寝たままプレイルームに行こう」
4. 「夕食後もお母さんに付き添ってもらおう」

回答番号 3
1. 根拠のない励ましは不安や不信につながる。
2. 無制限の視聴ではストレス対処として不十分である。
3. 安静を守りながら遊びや気分転換を取り入れる支援が適切である。
4. 付き添い延長だけでは児の活動制限によるストレスに対応できない。

70. 流行性耳下腺炎で髄膜炎疑いの5歳男児。退院後の注意として父親への回答で適切なのはどれか。
1. 「Aちゃんの精巣の腫れに注意してください」
2. 「Aちゃんは退院後1週間は登園できません」
3. 「Aちゃんの耳の聴こえ方に注意してください」
4. 「AちゃんからBちゃんへの感染予防には明日までのワクチン接種が効果的です」

回答番号 3
1. 精巣炎は思春期以降で問題になりやすく、5歳では最優先ではない。
2. 登園停止は「耳下腺などの腫脹出現後5日を経過し、全身状態が良好」などを基準に判断する。
3. 流行性耳下腺炎では難聴が合併することがあり、聴こえ方に注意する。
4. 曝露後のワクチン接種で確実な感染予防効果は期待しにくい。
