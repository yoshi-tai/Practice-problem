## 問題2　アポトーシスで正しいのはどれか。

1. 群発的に発現する。
2. 壊死のことである。
3. 炎症反応が関与する。
4. プログラムされた細胞死である。

**回答：4**

**解説**

1. 群発的に発現する。→ 孤発的に発現する。
2. 壊死のことである。→ 壊死ではない。
3. 炎症反応が関与する。→ 通常、炎症反応は関与しない。
4. プログラムされた細胞死である。→ 正しい。

## 問題5　創傷治癒について、一次治癒と比較した二次治癒の特徴はどれか。

1. 瘢痕を形成する。
2. 組織欠損が少ない。
3. 肉芽組織量は少ない。
4. 組織修復は速やかである。

**回答：1**

**解説**

1. 瘢痕を形成する。→ 正しい。
2. 組織欠損が少ない。→ 二次治癒では組織欠損が多い。
3. 肉芽組織量は少ない。→ 二次治癒では肉芽組織量が多い。
4. 組織修復は速やかである。→ 二次治癒では修復は遅延しやすい。

一次治癒は切創のように創縁が近く、肉芽形成が少なく、早期にほぼ元通りに治癒する。

## 問題6　急性炎症と比較して慢性炎症に特徴的な所見はどれか。2つ選べ。

1. 好中球浸潤
2. CRPの上昇
3. リンパ球浸潤
4. 形質細胞の浸潤
5. 血管透過性の亢進

**回答：3、4**

**解説**

1. 好中球浸潤：急性炎症で特徴的。
2. CRPの上昇：急性期にも慢性期にもみられる。
3. リンパ球浸潤：慢性炎症で特徴的。
4. 形質細胞の浸潤：慢性炎症で特徴的。
5. 血管透過性の亢進：急性炎症で特徴的。

## 問題7　ウイルスが原因で発症するのはどれか。

1. 血友病
2. 鉄欠乏性貧血
3. 再生不良性貧血
4. 成人T細胞白血病〈ATL〉

**回答：4**

**解説**

1. 血友病：遺伝子病。伴性劣性遺伝で、第VIII因子または第IX因子の欠損による。
2. 鉄欠乏性貧血：鉄欠乏による貧血。
3. 再生不良性貧血：原因不明のことが多い。
4. 成人T細胞白血病〈ATL〉：ヒトTリンパ球向性ウイルス1型〈HTLV-1〉による。

## 問題8　糖尿病の診断指標となるのはどれか。

1. 尿酸値
2. HbA1c
3. 血液沈降速度
4. プロトロンビン時間

**回答：2**

**解説**

1. 尿酸値：高尿酸血症の指標。
2. HbA1c：長期的な血糖コントロールや糖尿病診断に使用される。
3. 血液沈降速度：炎症や感染などの指標。
4. プロトロンビン時間：凝固機能の指標。

## 問題9　糖尿病の三大合併症による症状はどれか。

1. う歯
2. 血尿
3. 昏睡
4. 失明
5. 吐血

**回答：4**

**解説**

1. う歯：糖尿病の三大合併症とは直接関係しない。
2. 血尿：糖尿病腎症では蛋白尿が主で、血尿は典型ではない。
3. 昏睡：糖尿病性ケトアシドーシスなどの急性合併症でみられる。
4. 失明：糖尿病性網膜症によって起こりうる。
5. 吐血：糖尿病の三大合併症とは直接関係しない。

**糖尿病の三大合併症**

- 糖尿病性網膜症：視力障害、失明
- 糖尿病性腎症：腎機能低下、慢性腎不全
- 糖尿病性神経障害：知覚障害、自律神経障害、壊疽

5. C型ウイルス性肝炎

**回答：5**

**解説**

4. 慢性腎不全による血液透析：糖尿病性腎症と関連する。
5. C型ウイルス性肝炎：糖尿病の進行による病態ではない。

## 問題11　黄疸や胆汁酸に関して正しいのはどれか。2つ選べ。

1. 胆汁酸は大部分が回腸から大腸で再吸収される。
2. 胆道癌による閉塞性黄疸〈肝後性黄疸〉では間接ビリルビンが増加する。
3. 高度な閉塞性黄疸では便が灰白色になる。
4. 溶血性黄疸〈肝前性黄疸〉では直接ビリルビンが増加する。
5. 間接ビリルビンは脾臓でグルクロン酸抱合されて直接ビリルビンに変換される。

**回答：1、3**

**解説**

1. 胆汁酸は大部分が回腸から大腸で再吸収される。→ 正しい。
2. 閉塞性黄疸では間接ビリルビンが増加する。→ 直接ビリルビンが増加する。
3. 高度な閉塞性黄疸では便が灰白色になる。→ 正しい。
4. 溶血性黄疸では直接ビリルビンが増加する。→ 間接ビリルビンが増加する。
5. 間接ビリルビンは脾臓で抱合される。→ 抱合は肝臓で行われる。

## 問題13　動脈硬化症の粥腫形成に関与するのはどれか。2つ選べ。

1. 血管内皮細胞
2. 肥満細胞
3. 泡沫細胞
4. メサンギウム細胞
5. 壁細胞

**回答：1、3**

**解説**

1. 血管内皮細胞：内皮障害が粥腫形成の開始点となる。
2. 肥満細胞：粥腫形成の中心的細胞ではない。
3. 泡沫細胞：マクロファージが脂質を取り込んだもので、粥腫の主成分となる。
4. メサンギウム細胞：腎糸球体の細胞。
5. 壁細胞：胃の細胞。

## 問題14　生活習慣病はどれか。

1. 髄膜炎
2. 虚血性心疾患
3. 関節リウマチ
4. アルツハイマー病

**回答：2**

**解説**

生活習慣病とは、食事、運動、喫煙、飲酒、ストレスなどの生活習慣が深く関与して発症する疾患の総称である。

代表例には、脳血管疾患、脳卒中、心疾患、心筋梗塞、虚血性心疾患、動脈硬化症、糖尿病、高血圧症、脂質異常症、肺癌、大腸癌などがある。

なお、2型糖尿病は生活習慣病に含まれるが、1型糖尿病は生活習慣病ではない。

## 問題16　浮腫の原因となるのはどれか。

1. 血漿膠質浸透圧の上昇
2. リンパ還流の不全
3. 毛細血管内圧の低下
4. 毛細血管透過性の低下

**回答：2**

**解説**

1. 血漿膠質浸透圧の上昇：低下すると浮腫が起こる。
2. リンパ還流の不全：浮腫の原因となる。
3. 毛細血管内圧の低下：上昇すると浮腫が起こる。
4. 毛細血管透過性の低下：亢進すると浮腫が起こる。

## 問題17　浮腫が生じやすいのはどれか。

1. 甲状腺機能亢進症
2. 過剰な運動
3. 低栄養
4. 熱中症

**回答：3**

**解説**

1. 甲状腺機能亢進症：浮腫は典型的ではない。甲状腺機能低下症では粘液水腫を起こす。
2. 過剰な運動：浮腫の主原因ではない。
3. 低栄養：アルブミン低下により膠質浸透圧が低下し、浮腫が起こる。
4. 熱中症：脱水が主体で、浮腫は一般的ではない。

## 問題27　貧血の定義で正しいのはどれか。

1. 血圧が下がること
2. 脈拍を自覚すること
3. 立ち上がると失神すること
4. 血色素量が減っていること

**回答：4**

**解説**

貧血とは、末梢血中のヘモグロビン〈Hb、血色素〉濃度、赤血球数、赤血球容積率〈ヘマトクリット、Ht〉が減少し、基準値未満になった状態である。

一般には、ヘモグロビン濃度が基準値を下回った場合をいう。成人の基準値は、男性で13.0g/dL未満、女性で12.0g/dL未満が目安である。高齢者では11.0g/dL未満を貧血の基準にすることもある。

## 問題29　鉄欠乏性貧血でみられる症状はどれか。2つ選べ。

1. 動悸
2. 発熱
3. 黄疸
4. 感覚過敏
5. 匙状爪

**回答：1、5**

**解説**

鉄欠乏性貧血では、以下のような症状がみられる。

- 動悸：少しでも酸素を運ぼうとして心拍数が増えるため。
- 息切れ：酸素を取り込もうとするため。
- 疲労感、倦怠感：筋肉への酸素供給が不足するため。
- 顔色不良
- 匙状爪：爪が反り返る、割れやすい、もろい。
- 味覚障害、舌のひりひり感

## 問題30　肺塞栓症について誤っているのはどれか。

1. 肥満が誘因となる。
2. 長期臥床が誘因となる。
3. 心電図所見は非特異的である。
4. 下肢よりも上肢の手術後に多い。
5. 深部静脈血栓症との合併が多い。

**回答：4**

**解説**

肺動脈血栓塞栓症の主な原因は、下肢にできた深部静脈血栓である。長期臥床、肥満、手術後などが誘因となる。上肢よりも下肢由来の血栓が重要である。

## 問題31　心原性ショックで直ちに現れる徴候はどれか。

1. 血圧の上昇
2. 体温の上昇
3. 尿量の増加
4. 脈拍数の増加

**回答：4**

**解説**

心原性ショックでは心拍出量が低下するため、代償的に少しでも酸素を運ぼうとして脈拍数が増加する。

- 血圧：低下する。
- 体温：低下することがある。
- 尿量：減少する。

## 問題32　出血性ショックで起こるのはどれか。

1. 体温の上昇
2. 尿量の増加
3. 血圧の低下
4. 皮膚の紅潮

**回答：3**

**解説**

ショックでは循環血液量や有効循環血液量が低下し、血圧が低下する。出血性ショックでは皮膚は紅潮ではなく蒼白になりやすく、尿量は減少する。

## 問題33　心外閉塞・拘束性ショックの原因として適切でないのはどれか。2つ選べ。

1. 肺血栓塞栓症
2. 緊張性気胸
3. 大動脈解離
4. 心筋梗塞
5. 心タンポナーデ
6. 敗血症

## 問題58　病理学的な悪性度が最も高いのはどれか。

3. 神経膠芽腫
